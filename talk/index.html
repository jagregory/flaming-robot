Media Queries

    background: green;
  }
}

  body {
    background: blue;
  }
}

Ors

  body {
    background: blue;
  }
}

Not

  }

Units

Who remembers EMs?

1em = font size

Traditionally 1em was the maximum width of the block used for a letter on a printing press. The letter M took up the full width of a block hence the name an EM.

Why are EMs good?

Relative

Why are EMs bad?

Relative to parent

Who's heard of REMs?

1rem = font size

of the root element

What about VWs?

1vw = viewport width / 1000

1vh = viewport height / 1000

1vmin = min(1vh, 1vm)

1vmax = max(1vh, 1vm)

Isn't that just %?

% is relative to a parent

Parent is sometimes the viewport, but not always
